# 개인정보 처리방침 (Privacy Policy)

**시행일: 2026년 1월 28일**

**HLS 비디오 다운로더(HLS Video Downloader)**(이하 "본 확장 프로그램")는 사용자의 개인정보를 소중하게 생각합니다. 본 프로그램은 `crx-media-extractor`와 동일한 운영 주체인 **SSEYLAB**에 의해 개발 및 관리되며, 동일한 수준의 개인정보 보호 원칙을 준수합니다.

---

### 1. 개인정보 수집 및 이용 (Core Principle)

본 확장 프로그램은 **사용자의 개인정보(IP 주소, 로그인 정보 등)나 브라우징 기록을 수집하여 영구 저장하거나 판매하지 않습니다.**

- 스트리밍 영상 다운로드 및 변환 작업을 위해 감지된 비디오 주소 정보가 **SSEY LAB이 제공하는 백엔드 API 서버**로 전송될 수 있습니다.
- 전송된 주소 정보는 오직 **영상 다운로드 및 포맷 변환(ffmpeg)** 작업만을 위해 일시적으로 사용되며, 작업 완료 후 시스템 로그를 제외한 원본 데이터는 파기됩니다.
- 서비스 운영 정책에 따라 백엔드 서버는 로컬 호스트(`localhost`) 또는 SSEY LAB이 관리하는 외부 서버에서 운영될 수 있습니다.
- 사용자 설정 및 기록은 브라우저의 로컬 저장소(`chrome.storage.local`)에만 저장됩니다.

### 2. 사용 권한 및 목적
원활한 스트리밍 영상 다운로드 기능을 제공하기 위해 아래와 같은 최소한의 권한을 사용합니다.

- **`activeTab`**: 현재 사용자가 보고 있는 탭의 제목(영상 제목 확인용)을 가져오기 위해 사용됩니다.
- **`webRequest`**: 네트워크 트래픽 중 `.m3u8` 스트리밍 미디어 주소만을 감지하여 목록을 생성하기 위해 사용됩니다. (감지된 주소는 로컬에만 저장됩니다.)
- **`storage`**: 감지된 영상 목록과 다운로드 옵션을 저장하여 사용자 편의성을 높이는 데 사용됩니다.
- **`downloads`**: 변환 완료된 MP4 파일을 사용자의 PC로 저장(다운로드)하기 위해 사용됩니다.

### 3. 데이터 처리 및 공유

본 확장 프로그램은 감지된 미디어 정보나 사용자 행동 데이터를 제3자에게 판매하거나 공유하지 않습니다. 

- 모든 데이터 처리는 확장 프로그램과 **SSEY LAB 백엔드 API 서버** 간의 통신으로 이루어집니다.
- 외부 서버를 이용하는 경우에도 데이터는 암호화된 통신을 통해 전송되며, **변환 처리 목적 이외의 용도로 사용되지 않습니다.**
