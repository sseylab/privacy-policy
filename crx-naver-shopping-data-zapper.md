# 개인정보 처리방침 (Privacy Policy)

**시행일: 2026년 2월 3일**

**네이버 쇼핑 목록 추출기(Naver Shopping Data Zapper)**(이하 "본 확장 프로그램")는 사용자의 개인정보를 소중하게 생각하며, 관련 법령을 준수하기 위해 최선을 다하고 있습니다. 본 방침은 본 확장 프로그램이 사용자의 데이터를 어떻게 취급하는지 투명하게 안내하기 위해 작성되었습니다.

---

### 1. 개인정보 수집 및 이용 (핵심 원칙)

본 확장 프로그램은 **사용자의 어떠한 개인정보(IP 주소, 로그인 정보, 검색어 등)나 브라우징 기록도 수집, 저장 또는 외부 서버로 전송하지 않습니다.**

- 본 서비스는 네이버 쇼핑 검색 결과 페이지에서 **공개된 상품 정보만을 추출**합니다.
- 추출된 상품 데이터(쇼핑몰명, 상품명, 평점, 리뷰수 등)는 사용자의 로컬 브라우저 내에서만 처리되며, 외부 서버로 전송되지 않습니다.
- 추출된 파일 목록이나 설정 정보는 확장 프로그램의 저장소(`chrome.storage.local`)에만 임시로 저장되며, 브라우저를 초기화하거나 확장 프로그램을 삭제하면 즉시 사라집니다.

### 2. 사용되는 권한 및 목적

본 확장 프로그램은 원활한 기능 제공을 위해 다음과 같은 권한을 사용하며, 이는 오직 명시된 목적을 위해서만 사용됩니다.

- **`activeTab`**: 현재 활성화된 네이버 쇼핑 페이지의 상품 요소(쇼핑몰명, 상품명, 평점 등)를 탐색하기 위해 사용됩니다.
- **`scripting`**: 웹페이지 내에서 상품 정보를 감지하는 로직을 실행하기 위해 사용됩니다.
- **`downloads`**: 추출된 상품 목록을 CSV 파일로 사용자의 컴퓨터에 안전하게 저장하기 위해 사용됩니다.
- **`storage`**: 사용자가 설정한 필터 옵션(자동 스크롤 여부, 추출 항목 수 등)을 브라우저 내에 기억하기 위해 사용됩니다.
- **`notifications`**: 상품 추출이 완료되었음을 사용자에게 알리기 위해 사용됩니다.

### 3. 호스트 권한

본 확장 프로그램은 다음 호스트에 대한 권한을 가지고 있습니다.

- **`https://search.shopping.naver.com/*`**: 네이버 쇼핑 검색 페이지에서 상품 정보를 추출하기 위해 필요합니다.
- **`https://www.google-analytics.com/*`**: Google Analytics를 통해 기능 사용성을 분석하기 위해 필요합니다. (수집 항목: 추출된 상품 수, 기능 사용 여부 등 최소한의 통계 정보만 수집)

### 4. 제3자 제공 및 위탁

본 확장 프로그램은 추출된 데이터를 외부 서버로 전송하지 않으므로, 사용자의 정보를 제3자에게 제공하거나 외부 업체에 처리를 위탁하는 행위를 일절 하지 않습니다.

**단, Google Analytics를 통한 통계 데이터는 제한된 범위 내에서 Google에 전송될 수 있습니다.**
- 전송 항목: 기능 사용 통계, 추출된 항목 수 등의 **익명 데이터만** 수집됩니다.
- 개인을 특정할 수 있는 정보(IP 주소, 검색어, 상품 정보 등)는 포함되지 않습니다.
