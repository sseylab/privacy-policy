# 개인정보 처리방침 (Privacy Policy)

**시행일: 2026년 1월 25일**

**CRX 미디어 추출기(CRX Media Extractor)**(이하 "본 확장 프로그램")는 사용자의 개인정보를 매우 소중하게 생각하며, 관련 법령을 준수하기 위해 최선을 다하고 있습니다. 본 방침은 본 확장 프로그램이 사용자의 데이터를 어떻게 취급하는지 투명하게 안내하기 위해 작성되었습니다.

---

### 1. 개인정보 수집 및 이용 (Core Principle)

본 확장 프로그램은 **사용자의 어떠한 개인정보나 브라우징 기록도 수집, 저장 또는 외부 서버로 전송하지 않습니다.** 

- 본 서비스는 모든 미디어 추출 작업을 사용자의 로컬 브라우저 내에서만 수행합니다.
- 추출된 파일 리스트나 설정 정보는 외부로 공유되지 않으며, 확장 프로그램의 저장소(local storage)에만 임시로 머무릅니다.

### 2. 사용되는 권한 및 목적

본 확장 프로그램은 원활한 기능 제공을 위해 다음과 같은 권한을 사용하며, 이는 오직 명시된 목적을 위해서만 사용됩니다.

- **activeTab**: 현재 활성화된 페이지의 미디어 요소(이미지, 동영상 등)를 탐색하고 주소를 추출하기 위해 사용됩니다.
- **scripting**: 웹페이지 내에서 미디어 파일을 감지하는 로직을 실행하기 위해 사용됩니다.
- **downloads**: 추출된 미디어 파일을 사용자의 컴퓨터에 안전하게 저장하기 위해 사용됩니다.
- **storage**: 사용자가 설정한 필터 옵션(검색어, 크기 제한 등)을 브라우저 내에 기억하기 위해 사용됩니다.

### 3. 제3자 제공 및 위탁

본 확장 프로그램은 데이터를 외부로 전송하지 않으므로, 사용자의 정보를 제3자에게 제공하거나 외부 업체에 처리를 위탁하는 행위를 일절 하지 않습니다.

### 4. 분석 및 트래킹

본 확장 프로그램은 Google Analytics와 같은 외부 분석 도구나 웹 트래커를 포함하지 않습니다.

### 5. 개인정보 처리방침의 변경

본 방침은 기능 개선이나 관련 법령의 변화에 따라 수정될 수 있습니다. 변경 사항이 있을 경우 본 페이지를 통해 공지하며, 상단의 시행일을 업데이트합니다.
