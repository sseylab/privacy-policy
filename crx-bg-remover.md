# 개인정보 처리방침 (Privacy Policy)

**시행일: 2026년 2월 8일**

**SSEY LAB - 배경 제거 다운로더 (Background Remover)**(이하 "본 확장 프로그램")는 사용자의 개인정보를 매우 소중하게 생각하며, 관련 법령을 준수하기 위해 최선을 다하고 있습니다. 본 방침은 본 확장 프로그램이 사용자의 데이터를 어떻게 취급하는지 투명하게 안내하기 위해 작성되었습니다.

---

### 1. 개인정보 수집 및 이용 (Core Principle)

본 확장 프로그램은 **사용자의 어떠한 개인정보나 브라우징 기록도 수집, 저장 또는 외부 서버로 전송하지 않습니다.**

- 본 서비스는 이미지의 배경 제거를 위해 **SSEY LAB이 제공하는 백엔드 API 서버**로 이미지를 전송하여 처리합니다.
- 전송된 이미지는 오직 배경 제거 작업만을 위해 일시적으로 사용되며, **처리 완료 후 즉시 파기**됩니다.
- 서비스 운영 정책에 따라 로컬 호스트(`localhost`) 또는 SSEY LAB이 관리하는 외부 서버에서 데이터가 처리될 수 있습니다.
- 사용자 설정 및 기록은 브라우저의 로컬 저장소(LocalStorage/Chrome Storage)에만 저장되며 외부로 전송되지 않습니다.

### 2. 사용되는 권한 및 목적

본 확장 프로그램은 원활한 기능 제공을 위해 다음과 같은 권한을 사용하며, 이는 오직 명시된 목적을 위해서만 사용됩니다.

- **activeTab**: 사용자가 보고 있는 현재 탭의 이미지에 접근하여 배경 제거 기능을 수행하기 위해 사용됩니다.
- **scripting**: 웹페이지 내 이미지 요소에 "배경 제거 다운로드" 버튼을 표시하고 동작하게 하는 스크립트를 주입하기 위해 사용됩니다.
- **downloads**: 배경이 제거된 결과물(PNG 파일)을 사용자의 컴퓨터에 저장하기 위해 사용됩니다.
- **storage**: 사용자의 환경 설정(기능 ON/OFF, 서버 주소 등)을 저장하여 유지하기 위해 사용됩니다.
- **Host Permissions (<all_urls>)**: 모든 웹사이트의 이미지에 접근하여 배경 제거 기능을 제공하기 위해 필요합니다.

### 3. 제3자 제공 및 위탁

본 확장 프로그램은 데이터를 외부로 전송하지 않으므로, 사용자의 정보를 제3자에게 제공하거나 외부 업체에 처리를 위탁하는 행위를 일절 하지 않습니다. 모든 데이터 처리는 사용자의 로컬 환경 내에서 이루어집니다.

### 4. 분석 및 트래킹

본 확장 프로그램은 Google Analytics와 같은 외부 분석 도구나 웹 트래커를 포함하지 않습니다.

### 5. 개인정보 처리방침의 변경

본 방침은 기능 개선이나 관련 법령의 변화에 따라 수정될 수 있습니다. 변경 사항이 있을 경우 본 페이지를 통해 공지하며, 상단의 시행일을 업데이트합니다.
