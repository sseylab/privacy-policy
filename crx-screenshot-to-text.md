# 개인정보 처리방침 (Privacy Policy)

**시행일: 2026년 2월 13일**

**SSEY LAB - 캡처 이미지에서 텍스트 추출 (Screenshot-to-Text OCR)**(이하 "본 확장 프로그램")는 사용자의 개인정보를 매우 소중하게 생각하며, 관련 법령을 준수하기 위해 최선을 다하고
있습니다. 본 방침은 본 확장 프로그램이 사용자의 데이터를 어떻게 취급하는지 투명하게 안내하기 위해 작성되었습니다.

---

### 1. 개인정보 수집 및 이용 (Core Principle)

본 확장 프로그램은 **사용자의 어떠한 개인정보나 브라우징 기록도 수집, 저장 또는 외부 서버로 전송하지 않습니다.**

- 본 서비스는 캡처된 이미지 내의 텍스트 추출(OCR)을 위해 **SSEY LAB이 제공하는 백엔드 API 서버**로 이미지 데이터를 전송하여 처리합니다.
- 전송된 이미지 데이터는 오직 OCR 분석만을 위해 일시적으로 사용되며, **분석 완료 후 즉시 파기**됩니다. 수집된 텍스트 역시 서버에 저장되지 않으며 사용자에게 즉시 반환됩니다.
- 서비스 운영 정책에 따라 로컬 호스트(`localhost`)에서 데이터가 처리될 수 있습니다.
- 사용자 설정 및 OCR 결과 데이터는 브라우저의 로컬 저장소(Chrome Storage)에만 저장되며 외부로 전송되지 않습니다.

### 2. 사용되는 권한 및 목적

본 확장 프로그램은 원활한 기능 제공을 위해 다음과 같은 권한을 사용하며, 이는 오직 명시된 목적을 위해서만 사용됩니다.

- **activeTab / tabs**: 사용자가 보고 있는 현재 탭의 영역을 캡처하기 위해 사용됩니다.
- **offscreen**: 캡처된 이미지 데이터를 효율적으로 처리하기 위한 백그라운드 프로세스 용도로 사용됩니다.
- **sidePanel**: OCR 결과 및 편집 도구를 사용자에게 편리하게 제공하기 위한 인터페이스 용도로 사용됩니다.
- **downloads**: 추출된 텍스트 데이터(TXT)를 사용자의 컴퓨터에 저장하기 위해 사용됩니다.
- **storage**: 사용자의 환경 설정 및 최근 캡처 데이터를 저장하여 유지하기 위해 사용됩니다.
- **scripting**: 웹페이지 상에서 캡처 영역을 선택할 수 있는 오버레이 UI를 표시하기 위해 사용됩니다.

### 3. 제3자 제공 및 위탁

본 확장 프로그램은 데이터를 외부로 전송하지 않으므로, 사용자의 정보를 제3자에게 제공하거나 외부 업체에 처리를 위탁하는 행위를 일절 하지 않습니다.

### 4. 분석 및 트래킹

본 확장 프로그램은 외부 분석 도구나 웹 트래커를 포함하지 않습니다.

### 5. 개인정보 처리방침의 변경

본 방침은 기능 개선이나 관련 법령의 변화에 따라 수정될 수 있습니다. 변경 사항이 있을 경우 본 페이지를 통해 공지하며, 상단의 시행일을 업데이트합니다.
